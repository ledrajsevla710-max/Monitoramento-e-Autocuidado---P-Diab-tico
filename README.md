# 👣 Passo Seguro: Monitoramento e Autocuidado do Pé Diabético

O **Passo Seguro** é um aplicativo interativo desenvolvido em Python com o framework **Streamlit**, focado na prevenção do pé diabético e na promoção do autocuidado. [cite_start]Este projeto faz parte do desenvolvimento de ferramentas tecnológicas para o **PPGBiotec**.

## 🎯 Objetivo
Auxiliar pacientes diabéticos no acompanhamento de sua saúde podológica, permitindo o registro de sintomas e fornecendo orientações educativas rápidas baseadas em protocolos de saúde para evitar complicações graves.

## 🚀 Funcionalidades
* **Perfil do Paciente:** Coleta de dados como nome, data de nascimento, idade, sexo e cidade para traçar o perfil do usuário.
* **Avaliação de Risco:** Questionário sobre o tempo de diagnóstico de diabetes.
* **Triagem Clínica:** Registro de presença de calosidades ou úlceras (feridas), com identificação do local afetado.
* **Histórico de Amputação:** Campo específico para informar amputações prévias e localização anatômica.
* **Flashcards Educativos:** Orientações sobre autocuidado diário, incluindo:
    * Inspeção visual diária dos pés.
    * Hidratação correta (evitando a região entre os dedos).
    * Uso de calçados adequados e proteção constante.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Framework:** Streamlit
* **Análise de Dados:** Pandas
