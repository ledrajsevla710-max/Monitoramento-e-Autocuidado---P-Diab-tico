Ele está estruturado para dar um ar profissional ao seu repositório, destacando o suporte ao projeto do PPGBiotec e as funcionalidades que implementamos.👣 Passo Seguro: Monitoramento e Autocuidado do Pé DiabéticoO Passo Seguro é um aplicativo interativo desenvolvido em Python com o framework Streamlit, focado na prevenção do pé diabético e na promoção do autocuidado. O projeto faz parte das etapas de desenvolvimento de ferramentas tecnológicas para o PPGBiotec.🎯 ObjetivoAuxiliar pacientes diabéticos no acompanhamento de sua saúde podológica, permitindo o registro de sintomas (calosidades, úlceras e amputações) e fornecendo orientações educativas rápidas baseadas em protocolos de saúde.🚀 FuncionalidadesPerfil do Paciente: Coleta de dados demográficos (nome, idade, sexo e cidade) para contextualização do atendimento.Triagem de Risco: Questionário dinâmico sobre tempo de diagnóstico e histórico de lesões.Mapa de Amputação: Interface visual para identificação de áreas afetadas por procedimentos cirúrgicos.Flashcards Educativos: Orientações visuais e diretas sobre inspeção diária, hidratação correta e escolha de calçados adequados.🛠️ Tecnologias UtilizadasLinguagem: PythonFramework Web: StreamlitProcessamento de Dados: Pandas / Numpy
